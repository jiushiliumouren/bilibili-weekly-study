# B站每周榜单 BV 号抓取工具

## 📊 项目概述

本项目是一个**端到端的数据分析练习项目**，旨在通过 B站“每周必看”榜单数据，完成从数据采集、存储到可视化分析的全流程。

### 当前阶段（已完成）
- 使用 Python + Selenium 抓取指定期数的 BV 号
- 自动导出为结构化 Excel 文件

### 后续规划
- **数据入库**：将 Excel 数据清洗后存入 **MySQL / PostgreSQL**，便于历史数据查询和管理
- **多期数据积累**：定期抓取多期榜单，构建时间序列数据集
- **数据增强**：通过 B站 API 获取每个 BV 的播放量、投币数、弹幕数等详细信息
- **可视化分析**：使用 **Power BI** 或 **Tableau** 制作仪表板，分析热门视频趋势、UP主表现、内容类别偏好等

### 技术栈
| 环节 | 工具 |
|------|------|
| 数据采集 | Python, Selenium, pandas |
| 数据存储 | SQL (MySQL/PostgreSQL) |
| 数据可视化 | Power BI / Tableau |
| 版本控制 | Git, GitHub |
根据期数（如 366 期）自动获取 Bilibili 每周必看中的所有 BV 号，并导出为 Excel 文件。

## 📦 安装

1. 克隆本项目
   ```bash
   git clone https://github.com/你的用户名/bilibili-weekly-bv.git
   cd bilibili-weekly-bv
   ```

2. 安装依赖（需要 Python 3.7+）
   ```bash
   pip install -r requirements.txt
   ```

3. 确保已安装 [Microsoft Edge 浏览器](https://www.microsoft.com/edge)

## 🚀 使用

## 📈 第一步：获取视频BV号

```bash
python main.py --week 366
```

可选参数：

| 参数 | 说明 |
|------|------|
| `--week` | 必填，期数，如 366 |
| `--output` | 输出目录，默认为当前目录 |
| `--headless` | 无头模式（不弹出浏览器窗口） |

示例：

```bash
# 指定输出到 data 文件夹，并启用无头模式
python main.py --week 366 --output ./data --headless
```
## 📈 第二步：获取视频详细信息

在获得 BV 号列表后，运行以下命令调用 B站 API 获取每个视频的播放量、UP主、互动数据等：

```bash
python fetch_video_details.py --input ./output/366bv.xlsx --output ./data/366.csv

参数说明：
--input ：第一步生成的 Excel 文件路径（包含 BV 号）
--output ：保存详细数据的 CSV 文件路径
--delay ：请求间隔（秒），避免请求过快
--bv-column：BV 号所在列（字母或数字），默认为 A
输出字段包括：标题、UP主、播放、点赞、投币、收藏、评论、弹幕数、分享数、视频时长、发布时间等。
```
